## CSS 변수 정의
```css
:root {
  /* Primary Colors */
  --color-primary: #1A1A1A;
  --color-primary-light: #333333;
  --color-primary-dark: #000000;

  /* Secondary Colors */
  --color-secondary: #5B9BD5;
  --color-accent: #F06292;

  /* Background Colors */
  --color-bg-primary: #F5F0E8;
  --color-bg-secondary: #1A1A1A;

  /* Text Colors */
  --color-text-primary: #1A1A1A;
  --color-text-secondary: #FFFFFF;
  --color-text-muted: #888888;

  /* Interactive Colors */
  --color-button-primary: #5B9BD5;
  --color-button-hover: #4A90D9;
  --color-link: #5B9BD5;
  --color-link-hover: #4A90D9;
}
```

## 컬러 사용 가이드

### Primary — 블랙 계열 (`#1A1A1A`)
- 페이지 주요 텍스트 (제목, 본문)
- 다크 섹션 배경 (푸터, 갤러리 영역)
- 로고 및 아이콘 기본 색상

### Secondary — 블루 계열 (`#5B9BD5`)
- 헤더/네비게이션 배경
- 버튼, 링크 등 상호작용 요소
- 배너, 띠 장식 요소

### Accent — 핑크 (`#F06292`)
- CTA(Call-to-Action) 강조 버튼
- 시선을 끌어야 하는 포인트 요소
- 배지, 태그, 알림 표시

### 보조 포인트 색상
| 색상 | HEX | 사용 용도 |
|------|-----|-----------|
| 옐로우 | `#FFD740` | 하이라이트, 세일 배지, 카드 배경 |
| 레드/코랄 | `#E84855` | 경고, 긴급 알림, 할인 태그 |

### Background — 크림 화이트 (`#F5F0E8`)
- 메인 콘텐츠 영역 배경
- 카드 컴포넌트 배경
- 순백(`#FFFFFF`)보다 따뜻하고 빈티지한 톤 유지

### Text — 가독성 기준
| 배경색 | 텍스트 색상 | 용도 |
|--------|------------|------|
| 크림 `#F5F0E8` | 블랙 `#1A1A1A` | 일반 본문 |
| 블랙 `#1A1A1A` | 화이트 `#FFFFFF` | 다크 섹션 텍스트 |
| 블루 `#5B9BD5` | 화이트 `#FFFFFF` | 배너, 버튼 텍스트 |
| 크림 `#F5F0E8` | 그레이 `#888888` | 부가 설명, 캡션 |

## 반응형 고려사항

### 다크 모드
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #1A1A1A;
    --color-bg-secondary: #2D2D2D;
    --color-text-primary: #F5F0E8;
    --color-text-secondary: #CCCCCC;
    --color-text-muted: #999999;
    --color-secondary: #7BB3E0;
    --color-accent: #F48AAF;
    --color-button-primary: #7BB3E0;
    --color-button-hover: #5B9BD5;
    --color-link: #7BB3E0;
    --color-link-hover: #5B9BD5;
  }
}
```

### 모바일 대응
- 작은 화면에서는 블루/핑크 등 강조색의 면적을 줄이고, 배경색 위주로 구성
- 터치 영역의 버튼은 `--color-button-primary`를 사용하되, 최소 44px 크기 확보
- 텍스트 대비율 WCAG AA 기준 (4.5:1) 이상 유지

### 접근성 대비율 검증
| 조합 | 대비율 | WCAG 등급 |
|------|--------|-----------|
| `#1A1A1A` on `#F5F0E8` | 14.5:1 | AAA |
| `#FFFFFF` on `#1A1A1A` | 16.7:1 | AAA |
| `#FFFFFF` on `#5B9BD5` | 3.2:1 | AA (Large) |
| `#FFFFFF` on `#F06292` | 3.5:1 | AA (Large) |
